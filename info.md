# JD Smart

Unofficial Home Assistant integration for devices connected through JD Smart /
JD Xiaojia.

Current support includes air-conditioner controls and a tested subset of MiniJ
washing-machine controls. Authentication can be refreshed automatically while
the JD session remains valid, or imported as one JSON value from the optional
Android/ADB capture helper.

See the repository README for installation, authentication, security, and
device compatibility details.
